## Installation

1. Install the [Tampermonkey](https://www.tampermonkey.net/) extension for your browser (Chrome, Firefox, Edge, etc.).
2. Click the **Install this script** button on GreasyFork.
3. Open any game page on `fitgirl-repacks.site` and the script will automatically do its work!
